# Memory Game

This a memory game challenge built with Vue3, Vite and Pinia.

### Clone repository
```sh
git clone https://github.com/mariovaldivia/memory-game.git
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### In web browser you can access in address

```sh
http://localhost:5173
```
